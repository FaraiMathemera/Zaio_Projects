# Authentication System Challenge


## Instructions:
```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5000
```

## Technologies:
- HTML
- CSS (Bootstrap)
- NodeJS (Passport, Express, Mongoose, ejs, bcrypt)
- MongoDB (Atlas Flavour, *had issues with 32bit install*)


## Features:
- Validation
- Encrypted passwords
